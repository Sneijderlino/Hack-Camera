# Contributing to Hack-Camera

Thank you for considering contributing to Hack-Camera! Contributions are
welcome from everyone — whether it's fixing bugs, improving documentation,
adding templates, or suggesting new features.

## How to contribute

1. Fork the repository and clone your fork locally.
2. Create a new branch for your change: `git checkout -b feature/your-topic`.
3. Make your changes and commit with clear messages.
4. Push your branch to your fork: `git push origin feature/your-topic`.
5. Open a pull request against the `main` branch of the original repo.

Please ensure your code follows the existing style and shell conventions.
Document new features in `README.md` and update this file if guidelines
change.

## Reporting issues

If you encounter bugs or have ideas for improvements, open an issue with as
much detail as possible, including steps to reproduce and environment details.

## Code of Conduct

By participating in this project, you agree to abide by a respectful and
inclusive code of conduct. Inappropriate behavior will not be tolerated.
